# Workflow Admin (4 étapes)

Interface à onglets dans `AdminPage.tsx` pour configurer un projet.

## Fichiers

| Fichier | Étape | Rôle |
|---------|-------|------|
| `ImportStep.tsx` | 1 | Upload image coloriage + vidéo animation |
| `TriangulationStep.tsx` | 2 | Éditeur maillage (voir `../triangulation/CLAUDE.md`) |
| `PdfStep.tsx` | 3 | Génération PDF avec maillage et marqueurs L |
| `OpticalFlowStep.tsx` | 4 | Pré-calcul suivi optique des points du maillage |

## Étape 1 — Import

- Upload image (PNG/JPEG) → `project.originalImageBlob`
- Upload vidéo (MP4/WebM) → `project.videoBlob`
- Sauvegarde avec `updateProject(project, ['image'])` ou `['video']`

## Étape 2 — Triangulation

- Intègre `TriangulationCanvas` (voir dossier `triangulation/`)
- Modes : contour manuel, points internes, auto-mesh
- Densité ajustable (1-10) pour l'auto-mesh
- `isAutoGenerated` ref pour permettre le re-calcul live au changement de densité

## Étape 3 — PDF

- Génère un PDF (jsPDF) avec l'image + overlay du maillage triangulé
- Inclut les marqueurs L aux 4 coins pour la détection au scan

## Étape 4 — Optical Flow (OpticalFlowStep.tsx)

### Pré-calcul

1. Appelle `precomputeOpticalFlow(videoBlob, meshPoints, imageW, imageH)`
2. Extraction frame par frame (24 FPS) via `<video>` + Canvas
3. Chaque frame envoyée au Worker OpenCV → Lucas-Kanade sparse
4. Résultat : `videoFramesMesh: Point2D[][]` (positions par frame)
5. Sauvegarde avec `updateProject(project, ['videoFramesMesh'])`

### Paramètres Lucas-Kanade (dans le Worker)

| Paramètre | Valeur |
|-----------|--------|
| Fenêtre recherche | 21×21 px |
| Niveaux pyramide | 3 |
| Critère arrêt | 30 itérations ou epsilon 0.01 |

### Prévisualisation

- Canvas jouant la vidéo à 24 FPS
- Overlay maillage animé avec points trackés
- Contrôles play/pause/rewind
- Affichage compteur de frame

### Pipeline de coordonnées

```
Points maillage (espace image)
  → Conversion vers espace vidéo (ratio dimensions)
  → Tracking Lucas-Kanade frame par frame
  → Reconversion vers espace image
  → Stockage dans videoFramesMesh
```

### Progression

Callback de progression avec étape courante et pourcentage, affiché dans l'UI.
